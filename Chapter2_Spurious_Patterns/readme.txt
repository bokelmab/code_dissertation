The folder "Forecasting" includes the code which was used for the forecasting competition.
The folder "Data_Preparation" includes code which was used to generate the modified GT versions.
"WrapperFunctions" includes basic functions which were used for different tasks.
"Data" includes the GT data as well as the arrival data.
"Dissimilarity" includes the code to calculate the dissimilarity measure
To run the code the .Rproj file needs to be opened with R studio: File>open project
